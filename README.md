<div align="center">
  <h1><code>Replacer</code></h1>
  <p>
    <strong>Changing image paths with Media Queries.</strong>
  </p>
</div>

## Usage

```
<script src="./replacer.umd.js"></script>
<script>
    replacer.init();
</script>
```

```
<img data-replace='[{"type":"tab","suffix":"_tab"}, {"type":"sp","suffix":"_tab"}]' src="imgpath/cat.png"  />
```
